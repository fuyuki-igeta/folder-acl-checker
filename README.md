# フォルダ権限確認ツール

Windows Server のフォルダ ACL（アクセス権）を再帰的に取得し、CSV で出力します。

---

## 動作要件

- Windows 10 / Windows Server 2016 以降
- Python 3.8 以上
- 管理者権限推奨（ないと一部フォルダの権限を取得できません）

---

## セットアップ

追加インストール不要です。Python が入っていればすぐ使えます。

---

## 使い方

```cmd
python acl_check_free.py C:\Share
```

実行すると `acl_report_YYYYMMDD_HHMMSS.csv` が同じフォルダに生成されます。

### オプション

| オプション | 説明 | 例 |
|-----------|------|-----|
| `--output ファイル名` | 出力先ファイル名を指定 | `--output report.csv` |

### 使用例

```cmd
:: 基本（カレントフォルダに CSV を出力）
python acl_check_free.py C:\Share

:: 出力先を指定
python acl_check_free.py C:\Share --output C:\work\report.csv

:: ネットワークドライブも可
python acl_check_free.py \\SERVER\Share
```

---

## 出力内容

| 列名 | 内容 |
|------|------|
| フォルダパス | 確認したフォルダのフルパス |
| アカウント | アクセス権を持つユーザー・グループ名 |

---

## 制限事項

- フォルダのみ対象（ファイルは対象外）
- アカウント名のみ出力（権限の種類・継承情報は出力されません）
- アクセス拒否されたフォルダは `[WARN]` を表示してスキップします

---

## 注意事項

- 本ツールは読み取り専用です。権限の変更は一切行いません。
- 実行前に対象フォルダへの読み取り権限があることを確認してください。
